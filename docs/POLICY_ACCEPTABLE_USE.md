# Acceptable Use Policy | سياسة الاستخدام المقبول

## 1. Scope | النطاق

**EN:** This policy applies to all employees, contractors, and authorized users of AZAD Intelligent Systems resources, including computers, networks, cloud services, and customer data access.
**AR:** تنطبق هذه السياسة على جميع الموظفين والمقاولين والمستخدمين المصرّح لهم بموارد شركة أزاد للأنظمة الذكية، بما في ذلك الحواسيب والشبكات والخدمات السحابية والوصول إلى بيانات العملاء.

## 2. Authorized Use | الاستخدام المصرّح

| Resource (EN) | المورد (AR) | Permitted Use | الاستخدام المصرّح |
|---------------|-------------|---------------|------------------|
| Company laptops | حواسيب الشركة | Work-related tasks, approved learning | مهام العمل، التعلم المُعتمد |
| Internet | الإنترنت | Research, communication, cloud services | البحث، التواصل، الخدمات السحابية |
| Email | البريد | Business correspondence | المراسلات التجارية |
| GitHub | GitHub | Code contribution, issue tracking | المساهمة بالكود، تتبع المشكلات |

## 3. Prohibited Use | الاستخدام الممنوع

### 3.1 Illegal Activities | الأنشطة غير القانونية

**EN:** No hacking, fraud, piracy, or distribution of illegal content. No use of company resources for cryptocurrency mining.
**AR:** لا اختراق أو احتيال أو قرصنة أو توزيع محتوى غير قانوني. لا استخدام موارد الشركة للتعدين بالعملات المشفرة.

### 3.2 Security Risks | مخاطر الأمان

**EN:** No sharing of passwords or API keys. No bypassing security controls (VPN, firewalls, RBAC). No installing unauthorized software or browser extensions. No connecting unknown USB devices to company hardware.
**AR:** لا مشاركة كلمات المرور أو مفاتيح API. لا تجاوز ضوابط الأمان (VPN، جدران الحماية، RBAC). لا تثبيت برمجيات أو إضافات متصفح غير مُصرّح بها. لا توصيل أجهزة USB غير معروفة بأجهزة الشركة.

### 3.3 Data Mishandling | سوء معالجة البيانات

**EN:** No copying customer databases to personal devices. No emailing tenant data to personal accounts. No posting screenshots containing customer data on social media. No using customer lists for personal business.
**AR:** لا نسخ قواعد بيانات العملاء إلى أجهزة شخصية. لا إرسال بيانات المستأجر بالبريد إلى حسابات شخصية. لا نشر لقطات شاشة تحتوي على بيانات العملاء على وسائل التواصل. لا استخدام قوائم العملاء للأعمال الشخصية.

### 3.4 Network Abuse | إساءة استخدام الشبكة

**EN:** No excessive streaming, gaming, or large file downloads unrelated to work. No running port scanners or penetration testing tools without authorization.
**AR:** لا بث أو ألعاب أو تنزيل ملفات كبيرة بشكل مفرط غير مرتبطة بالعمل. لا تشغيل ماسحات المنافذ أو أدوات اختبار الاختراق دون تصريح.

### 3.5 Harassment | التحرش

**EN:** No sending offensive, discriminatory, or threatening messages. No using company communication channels for personal disputes.
**AR:** لا إرسال رسائل مسيئة أو تمييزية أو تهديدية. لا استخدام قنوات تواصل الشركة للنزاعات الشخصية.

## 4. Monitoring | المراقبة

**EN:** AZAD reserves the right to monitor: network traffic; email and Slack communications for compliance and security; access logs to production systems. Monitoring is for security, compliance, and operational integrity — not for micromanagement.
**AR:** تحتفظ أزاد بالحق في مراقبة: حركة المرور على الشبكة؛ اتصالات البريد و Slack للامتثال والأمان؛ سجلات الوصول إلى أنظمة الإنتاج. المراقبة للأمان والامتثال وسلامة العمليات — وليس للإدارة الدقيقة.

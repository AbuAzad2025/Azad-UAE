نظام الصلاحيات

يتحكم النظام بـ 4 أدوار رئيسية. كل دور له صلاحيات محددة لضمان الأمان والتنظيم.

Owner - المالك

صلاحيات كاملة
الصلاحيات الكاملة:
  • ✅ كل شيء بدون قيود
  • ✅ الوصول لـ Owner Mode
  • ✅ إدارة المستخدمين والأدوار
  • ✅ إدارة قاعدة البيانات
  • ✅ تصدير واستيراد البيانات
  • ✅ عرض وتعديل كل شيء
  • ✅ إدارة النظام والإعدادات
  • ✅ الوصول للبطاقات المحفوظة
تحذير: لا تمنح هذا الدور إلا لمن تثق به تماماً!

Super Admin - مدير عام

إدارة عالية
الصلاحيات:
  • ✅ إدارة المبيعات والمشتريات
  • ✅ إدارة الزبائن والمنتجات
  • ✅ إدارة المدفوعات والمخزون
  • ✅ عرض جميع التقارير
  • ✅ إدارة المصروفات
  • ✅ عرض دفتر الأستاذ
  • ✅ إلغاء وتعديل الفواتير
  • ❌ لا يمكنه الوصول لـ Owner Mode
الدور المناسب لمدير الفرع أو المدير التنفيذي

Manager - مدير

إدارة متوسطة
الصلاحيات:
  • ✅ إنشاء وعرض المبيعات
  • ✅ إنشاء وعرض المشتريات
  • ✅ إدارة الزبائن والمنتجات
  • ✅ إنشاء المدفوعات
  • ✅ عرض بعض التقارير
  • ✅ إدارة المخزون
  • ❌ لا يمكنه حذف المستخدمين
  • ❌ لا يمكنه عرض دفتر الأستاذ
الدور المناسب لمدير المخزون أو مدير المبيعات

Seller - بائع

صلاحيات محدودة
الصلاحيات:
  • ✅ إنشاء فواتير بيع فقط
  • ✅ عرض فواتيره الخاصة فقط
  • ✅ البحث عن المنتجات والزبائن
  • ✅ طباعة الفواتير
  • ❌ لا يمكنه حذف أو إلغاء
  • ❌ لا يمكنه عرض التقارير
  • ❌ لا يمكنه إدارة المخزون
  • ❌ لا يمكنه عرض أسعار التكلفة
الدور المناسب للبائعين والموظفين الميدانيين

معايير الأمان

✅ ما يُسمح به:
  • كل مستخدم يرى بياناته فقط (ما لم يكن مديراً)
  • البائع لا يرى أسعار التكلفة
  • البائع لا يرى أرباح الفواتير
  • المدير يرى تقارير محدودة
  • فقط Owner يرى البطاقات المحفوظة
❌ ما لا يُسمح به:
  • البائع لا يمكنه حذف أو إلغاء
  • البائع لا يمكنه تعديل الأسعار
  • المدير لا يمكنه إدارة المستخدمين
  • Super Admin لا يمكنه الوصول لـ Owner Mode
  • لا أحد يمكنه حذف بيانات Owner

إرشادات الاستخدام

📝 نصائح مهمة:
  1. لا تمنح Owner إلا لشخص واحد موثوق
  2. استخدم Super Admin للمدير التنفيذي
  3. استخدم Manager لمدراء الأقسام
  4. استخدم Seller للبائعين العاديين
  5. راجع الصلاحيات دورياً
  6. غيّر كلمات المرور الافتراضية فوراً
  7. عطّل المستخدمين غير النشطين بدلاً من حذفهم
